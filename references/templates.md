# 交付物模板

四个模板：转行档案（过程文件）+ 三份最终交付物。全部Markdown输出。生成时用用户的真实信息填充，删掉不适用的部分。

## 模板一：转行档案（阶段一创建，全程更新）

```markdown
# 转行档案 ｜ {用户称呼} ｜ 更新于 {日期}

## 当前进度
- [ ] 阶段一：获取信息（资料✅/方向✅/JD✅/测评✅）
- [ ] 阶段二：能力补齐与实操
- [ ] 阶段三：产出与面试准备

## 背景摘要
行业/岗位/年限：
在职状态：
核心技能与量化成绩：
协作过的角色：

## 方向决策
主投岗位：{X行业} AI产品经理（软件/硬件 ｜ B端/C端 ｜ 中小厂优先）
敲门砖备选：AI训练师 / AI运营
决策理由：

## 目标JD（2-3条，存全文）

## AI水平定级：L{n}
依据：
差距清单（按影响排序）：
1.
2.

## JD↔日常工作映射表
| JD职责 | 现有经验 | 差距 |

## 实操项目记录（阶段二）

## 下次对话从这里继续：
```

## 模板二：学习路径

```markdown
# {称呼}的AI转行学习路径 ｜ {X行业} AI产品经理

> 最终交付物：{一个可在面试现场演示的工作流/Agent/方案书} + 定制简历 + 面试题目单与速记卡
> 起点：L{n}（{一句话定级描述}） ｜ 预估周期：{n}周

## 第一程：认知补齐（第1-{n}周）
| 知识点 | 为什么学（对应JD哪条） | 看什么 | 动手做什么 | 学会的标志 |

## 第二程：实操项目（第{n}-{n}周）
项目名称：
解决的真实问题：
工具选型：{Coze/Dify/Claude...}，理由：
MVP范围（第一版只做什么）：
量化目标：
里程碑：①跑通最小流程 → ②真实/模拟数据验证 → ③效果数据沉淀 → ④STAR文档

## 第三程：面试冲刺（第{n}周起）
- [ ] 演示物定稿（30秒版/5分钟版讲法）
- [ ] 简历定稿并逐条对齐JD
- [ ] 题目单全部过一遍（先自答后打磨）
- [ ] 模拟面试至少1轮
- [ ] 体验目标公司产品，准备2个问题+1个建议

## 入职后前90天补什么（前瞻）
```

## 模板三：定制简历

```markdown
# {姓名}
{电话} ｜ {邮箱} ｜ {城市} ｜ 求职意向：{与JD职位名一致}

## 个人概述
{三标签公式：懂X行业业务（n年） + 有AI实操项目（项目名+效果数字） + 持续学习AI（具体在学什么）}

## AI实践项目
**{项目名}**（{时间}）
- 背景：{一句话痛点}
- 方案：{用了什么工具搭了什么，关键设计}
- 成果：{量化数字，对齐JD关键词}

## 工作经历
**{公司} ｜ {岗位}**（{时间}）
- {成就动词+做了什么+量化结果}（表述向JD关键词靠拢，如"搭建知识库""效果迭代""数据驱动"）

## 教育背景 / 技能
{学历} ｜ 工具：{Coze/Dify/Claude/数据工具...只写真会的}
```

注意：生成简历前逐条核对目标JD要求，确保每条核心要求在简历中有对应；每句话经用户确认真实。
