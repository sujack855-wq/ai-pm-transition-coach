# 面试题库生成规则与结构模板

本文件定义题目单的出题规则。结构源自真实AI岗面试的复盘沉淀，分A/B/C三部分。所有题目必须**基于用户的目标JD和真实经历定制**，以下题目是结构示范和兜底素材，不是照抄清单。

## A部分：自我介绍与项目概览（必备，要求极度清晰详细）

这部分是每场面试的开场，也是用户唯一能100%提前准备的部分，必须打磨到脱稿流畅。

**A1. 自我介绍（90秒版 + 3分钟版各一套）**

结构公式：我是谁（行业+年限）→ 三标签（"我懂X行业业务、我有AI实操项目、我在主动学AI"）→ 一个钩子成绩（最亮的量化数字）→ 为什么来面这个岗（与JD的契合点收尾）。

**A2. 项目STAR全流程介绍**

> 以一个具体项目为例，介绍你从接手到交付的全流程。

直接使用阶段二的STAR文档。追问预埋（用户必须准备好）：

- 这个方案/规则/流程设计的依据是什么？如果业务方需求不明确，你怎么办？
- 数据量/范围怎么确定的？标准是什么？
- 质量怎么把控的？（抽检比例、规则迭代周期这类具体数字要答得出）
- 遇到模糊的badcase怎么判断？
- 为什么是你来做这个项目？

**A3. 加分准备（强烈建议，性价比最高）**

- 体验过目标公司的AI产品后，你发现了什么问题？
- 你认为它目前最大的挑战是什么？你会怎么改进？

要求用户面试前真的去体验目标公司产品（试用智能客服、下载App），准备2个具体问题+1个改进建议。

## B部分：JD强相关模块化提问（核心区分度所在）

**生成规则**：把目标JD的职责逐条拆开，**每条职责生成一个模块，每模块2-3题**：1道经历题（"你有没有做过……具体怎么做的？"）+ 1道场景题（"如果遇到……你会怎么处理？"）+ 可选1道工具题（"你用过哪些……平台/工具？"）。

示范（假设JD是"电商智能客服运营/产品"）：

**模块：话术库/知识库建设**
- 你有没有搭建过话术库或知识库？具体怎么做的？
- 你们用什么智能客服平台？店小蜜、京小智这类平台熟吗？
- 大促期间话术逻辑变化很大，你怎么处理？

**模块：模型训练与效果迭代**
- 你怎么做机器人会话质检？发现问题之后怎么处理？
- 机器人自助解决率一直提不上去，你会怎么排查？（期望：分层归因——知识库覆盖？意图识别？话术质量？转人工策略？）

**模块：数据驱动**
- 你怎么输出效能报告？报告里有什么指标？
- 你用哪些工具做数据分析？

**模块：评测/标注类JD专属**（目标含AI训练师岗时使用）
- 在智能客服Agent场景下，评测应该关注哪些维度？
- 怎么判断一条训练数据的文本质量？
- 标注维度制定的依据是什么？评测集数据量怎么定？
- 发现标注员存在系统性理解偏差，怎么处理？

**模块：跨团队协作与业务理解**
- 你对我们公司了解多少？你觉得我们这个品类的AI应用有什么特殊挑战？
- 你怎么和客服团队/技术团队协作推动AI落地？

## C部分：软性问题（一题都不能裸奔）

全部要提前过一遍，重点的几道写进速记卡：

**动机与跨度类（转行者必被问，最高优先级）**
- 你之前做X行业，我们是Y，跨度这么大，你能快速上手吗？（标准应对：承认差异→提炼可迁移能力→用阶段二Demo证明学习速度→表达投入决心）
- 为什么想转AI/为什么想做这个岗位？怎么理解这个岗位？
- 为什么离职？（红线：不抱怨前司；公式：客观原因一句话+主动选择的成长叙事）

**自我认知类**
- 最喜欢/最不喜欢的工作内容？
- 过去做得好和不好的地方，反思和改善？
- 过去收获里，哪些可以复用到这个岗位？
- 1-2年的规划是什么？（要贴着这个岗位的成长线说，别说"我想做管理"）

**抗压与行为类**
- 之前工作里最沮丧的事？怎么处理的？
- 工作中遇到难应付的问题/不合理的需求，怎么解决？
- 举一个收到负面反馈的例子，你怎么应对的？学到了什么？
- 怎么看待压力？怎么舒缓？
- 记不住业务知识导致错误率高，怎么办？（期望：建立个人知识库/检查清单等系统性方法，而非"多努力"）
- 多任务并行怎么解决？
- 你怎么快速学习一个新领域？（最好直接拿"我用AI学AI"的真实过程作答）

**性格测评提醒**：很多公司面试前要求填性格测评和登记表。提醒用户：性格测评保持一致性即可（同维度的题前后答案别矛盾），不要刻意扮演"完美人格"，测评和面试表现差异过大反而扣分。

## 速记卡生成格式

题目单定稿后，为每道重点题生成一张速记卡：

```
【题】为什么离职？
【关键词】不抱怨 | 主动选择 | 成长叙事
【要点】前司业务收缩是事实，但重点讲：看到AI机会，已用3个月做出XX项目，是主动奔向而非被动逃离。
```

速记卡控制在15-25张（A部分3-5张、B部分按模块5-10张、C部分5-8张），按"面试当天候场10分钟能扫完"的密度来写。
