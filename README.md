# AI转行教练 Skill

一个跑在 WorkBuddy / Claude 里的 AI 转行教练，专门帮普通职场人和应届生转行 AI 产品经理。

不是给你一篇攻略——而是像真人教练一样，**三阶段陪你走完整个过程**，最后交付三样东西：

- ✅ 一份贴合目标 JD 的学习路径（终点是能演示的真实项目）
- ✅ 一份极度定制的简历
- ✅ 一份面试题库 + 速记卡

---

## 安装方法（30 秒）

**第一步**：复制这个仓库的链接

**第二步**：打开 WorkBuddy，把链接发给它，说：

```
帮我安装这个 SKILL：https://github.com/sujack855-wq/ai-pm-transition-coach
```

**第三步**：安装好之后，直接说：

```
我想转行 AI
```

它就会开始引导你了。

---

## 它会带你做什么

### 阶段一：摸底
- 收集你的背景和工作经历
- 帮你定转行方向（AI PM / AI训练师 / AI运营）
- 引导你找真实 JD，测出你的 AI 水平（L0–L4）

### 阶段二：补能力 + 动手做
- 按 JD 差距制定学习计划，只学有用的
- 带你在现有工作里找 AI 切入点，做出一个能演示的真实项目

### 阶段三：出简历 + 面试准备
- 贴 JD 逐条定制简历
- 生成面试题库，陪你模拟压测

---

## 适合谁

- 做电商 / 运营 / 教培 / 设计 / 客服，想往 AI 方向走的
- 校招没上岸，想去 AI 行业的应届生
- 觉得自己"不懂 AI"、不知道从哪开始的

**不适合**：纯技术岗（算法工程师等）的求职备考。
